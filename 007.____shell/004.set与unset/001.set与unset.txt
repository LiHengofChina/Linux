

//====================================== set 的作用：
//====================================== set 的作用：


设置变量或选项，但它不是专门用来定义变量的（定义变量直接赋值即可）。

//======= 示例：
//======= 示例：

VAR="Hello"  # 定义变量
set +x        # 关闭调试模式
set -x        # 打开调试模式


//====================================== unset 的作用：
//====================================== unset 的作用：

删除变量或函数，让它从当前环境中移除。

//======= 示例：
//======= 示例：
VAR="Hello"
unset VAR  # 删除变量 VAR
