


（1）创建一个交换文件
	sudo fallocate -l 16G /home/liheng/workspace_work_local/swapfile
	或
	sudo dd if=/dev/zero of=/home/liheng/workspace_work_local/swapfile bs=1M count=4096

（2）设置文件权限
	sudo chmod 777 /home/liheng/workspace_work_local/swapfile

（3）设置交换空间
	sudo mkswap /home/liheng/workspace_work_local/swapfile

（4）启用交换空间
	sudo swapon /home/liheng/workspace_work_local/swapfile


（5）使交换文件永久

		sudo echo "/home/liheng/workspace_work_local/swapfile swap swap defaults 0 0" >> /etc/fstab

（6）显示交换空间:  sudo swapon --show


	NAME                                       TYPE SIZE USED PRIO
	/swapfile                                  file 1.1G   0B   -2
	/home/liheng/workspace_work_local/swapfile file  16G   0B   -3
	
    //两个交换文件都可以同时有效，只要它们都已经被激活并在/etc/fstab文件中列出，系统会使用它们。

（7）查看交换空间，有没有被使用
	free -h
	//查看总的、已使用的以及可用的内存和交换空间。
